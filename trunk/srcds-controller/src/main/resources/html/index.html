Source Dedicated Server Controller @ ${hostname}

  • start: Starts the SRCDS instance
  • stop: Stops the SRCDS instance
  • status: Provides information on the current state of the SRCDS instance
  • showServerConfig: Shows the game-specific configuration of the SRCDS instance
  • showConfig: Shows the controller configuration
  • setConfig: Sets a configuration key to a specified value (e.g. setConfig?key=value)
  • editConfigFiles: Provides an interface to edit the game-specific configuration files
  • shutdown: Shuts the controller down
